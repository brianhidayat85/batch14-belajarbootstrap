Diberikan nama dan nilai mahasiswa, jika nilai tersebut lebih besar atau sama dengan 60 maka mahasiswa tersenut dinyatakan lulus, jika nilai lebih kecil dari  60 maka dinyatakan tidak lulus.

A. Naratif / Deskriptif
1. Mulai 
2. User memasukkan nama dan nilai 
3. Sistem menyimpan data nama dan nilai
4. Jika nilai lebih besar dari atau sama dengan 60, maka mahasiswa dinyatakan "Lulus"
5. Sedangkan jika nilai lebih kecil dari 60, maka mahasiswa dinyatakan "Tidak Lulus"
6. Sistem menampilkan nama dan keterangan kelulusan mahasiswa
7. Selesai 

B. Diagram Flowchart (Lihat Drawio)

C. Pseudo-code 
Algoritma_Kelulusan_Mahasiswa

// deklarasi dan inisialisasi nilai (input)
nama, keterangan: string 
nilai: float

// (pilih salah satu: input dari user / dari program)
// jika dari inputan user
print "Masukkan nama: "
get (nama)
print "Masukkan nilai: "
get (nilai)

// jika dari program 
// nama <-- "Rizki"
// nilai <-- 90

// proses / implementasi

// jika TRUE, maka eksekusi yang setelah THEN; jika FALSE, maka eksekusi setelah SELSE
IF nilai >= 60 THEN             // bisa TRUE atau FALSE

    keterangan = "Lulus"
ELSE 
    keterangan = "Tidak Lulus"
END IF 

// output 
print "Nama = " + nama
print "Keterangan = " + keterangan
